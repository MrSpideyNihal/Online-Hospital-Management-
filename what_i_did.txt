HOW TO USE THE HOOKS (EXAMPLE)
============================================================

Here's a quick example of how to wire a page to Supabase:

BEFORE (demo data):
```tsx
const DEMO = [{ id: '1', name: 'Rahul' }, ...]
// ... renders DEMO array
```

AFTER (real Supabase data):
```tsx
import { useAuth } from '@/lib/auth-context'
import { usePatients, useCreatePatient } from '@/lib/supabase/hooks'
import { toast } from 'sonner'

export default function PatientsPage() {
  const { hospitalId } = useAuth()
  const { data: patients, isLoading } = usePatients(hospitalId)
  const createPatient = useCreatePatient()

  const handleAdd = async (formData) => {
    try {
      await createPatient.mutateAsync({
        hospital_id: hospitalId!,
        full_name: formData.name,
        phone: formData.phone,
        // ... other fields
      })
      toast.success('Patient added!')
    } catch (err) {
      toast.error('Failed to add patient')
    }
  }

  if (isLoading) return <div>Loading...</div>

  return (
    // ... render patients (from `patients` variable, not DEMO)
  )
}
```
